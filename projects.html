Flower Website

Petal & Stem.

It's a static flower website.

A clean, semantic website for a local flower studio. It includes hero section, about page, and a simple contact area. Focuses on structure and telling a warm brand story, all with just html.

Netflix clone

Netflix homepage.

A basic clone of the netflix landing page layout. It includes header with navigation, hero section with call-to-action, and a grid-style card layout for movies/tv shows . I built to practice complex html structures.

NGO website

Hope Initiative

A simple website for a fictional non-profit organization working in Kenya. it includes mission statement, programs listing, volunteer section, and donation call-to-action. I built it with accessibility in mind.

Recipe Planner

Meal Planner Concept

A recipe planning tool concept that helps users organize meals.It includes a recipe list, ingredients table and weekly planner layout.It solves the "what's for dinner?" problem.

E-commerce Website

Urban Cart.

A basic e-commerce site structure with product cards, category sidebar, and a simple shopping cart layout . It focuses on product display and navigation.

Airbnb Website

PlaceFinder

A simplified clone of an airbnb-style listing page.It includes search bar, property cards with details and a basic footer.I built it to practice card layouts and listing structures.

Restaurant Website

A complete restaurant website with menu page, about section, reservation form, and contact information. I built to practice multiple page structures and form elements.
